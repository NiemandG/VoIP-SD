Поиск свободных extension

Зайти в CUCM, выполнить поиск в желаемом диапазоне добавочных, нажать CRTL+A (скопировать всё), вставить в окно на странице "Сервисы - поиск свободных extension" и нажать кнопку. Выполнится поиск существующих номеров на этой странице, а затем пропусков между ними.

Привязка extension

На странице "Сервисы - привязка extension" в первое окно вставить свободные добавочные (можно найти на странице "Сервисы - поиск свободных extension"), во второе вставить предварительно подготовленный список МАС-адресов, и нажать кнопку "Выполнить". Будет сформирован файл CSV стандартного формата, для массовой привязки МАС в CUCM.

Поиск пропущенных значений

Будет выполнен поиск пропущенных элементов последовательности. Удобно использовать для поиска свободных Finesse ID, полученных запросом "Выборка имеющихся Finesse ID" из раздела "Запросы SQL".

Проверка номера телефона

Определение принадлежности мобильного либо городского номера к оператору и региону. Переносы номеров между операторами не определяет, ниже результата выдаётся отдельная ссылка для проверки. Сервис предоставляется сторонним сайтом phoneverify.org.

Чистка МАС на CUCM

В базе SQL на AW выполнить поиск добавочных (запросом из раздела "Запросы SQL - выборка актуальных добавочных номеров для чистки МАС на CUCM"), которые совершали звонки за определённый период (в среднем достаточно двух месяцев). Получившийся результат - то что удалять нельзя. Удалять необходимо то чего здесь нет, т.е. пропуски. Находим их в разделе "Сервисы - поиск пропущенных значений", и вставляем в первое окно в разделе "Чистка МАС на CUCM". Далее идём на CUCM, ищем номера интересующего пула, выставив отображение 250 значений. Копируем, вставляем в notepad. Переходим на следующую страницу в CUCM, копируем, вставляем следом в то же окно notepad. Повторять до конца пула. Сохраняем файл с расширением txt, получается список всех страниц из CUCM, содержащий все текущие добавочные номера выбранного пула. Данный файл загружаем на той же странице "Чистка МАС на CUCM", и нажимаем кнопку "Выполнить". Далее в загруженном файле система выполняет поиск каждого из введённых добавочных для удаления, если находит то проверяет статус, если он не "Registered" и "Unregistered" (то есть "None"), то добавляет строку из файла, содержащую найденный добавочный в список. Далее из каждого элемента этого списка извлекается МАС-адрес и формируется CSV-файл стандартного формата, который загружается в CUCM для массового удаления добавочных номеров.
